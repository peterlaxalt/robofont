Theme Manager

By Connor Davenport and Andy Clymer

Theme Manager is an extension for writing,reading, and storing .roboFontTheme files. Each file uses RoboFont's native preferences to export and share user's preview color themes.

Features:
New Theme (✚)
 Writes new theme based on user's current settings
Delete Theme (✘)
 Removes selected theme from library
Duplicate Theme (❏)
 Duplicates selected theme into library
Import Theme
 Imports external .roboFontTheme file
Export Theme
 Writes .roboFontTheme file
Edit Theme (✑)
 Previews workspace to edit the current selected theme

Notes:
This preview displays *most* color preferences that the theme files support, but don't worry you can always apply the theme and then use the undo function if you don't like it.
